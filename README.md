# Corporate Finance Portfolio

This repository documents coursework, financial analysis, and project deliverables developed throughout BUS-629 International Corporate Finance. It serves as a structured portfolio that captures not only final outputs, but also the process used to create them, including documentation, financial models, validation work, and decision records.

The purpose of this repository is to maintain transparency, organization, and reproducibility across all project stages. Rather than storing isolated files, the repository follows a workflow-based structure that reflects how financial analysis and business decisions are developed in practice.

## Repository Objectives

This repository is designed to:

- Document work completed throughout BUS-629
- Organize financial models and supporting materials
- Record assumptions and major decisions
- Preserve analysis and validation processes
- Present polished deliverables in a professional format

## Workflow

Project work generally follows the process below:

1. Define objectives and assumptions
2. Gather supporting documentation and data
3. Build financial models
4. Validate calculations and outputs
5. Produce presentation-ready deliverables
